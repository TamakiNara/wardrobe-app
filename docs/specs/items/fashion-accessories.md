# ファッション小物の形・補助情報メモ

## 現状

### 種類

- `hat` / `belt` / `scarf_stole` / `gloves` / `jewelry` / `wallet_case` / `hair_accessory` / `eyewear` / `watch` / `other` を扱う

### 画面表示

- ファッション小物は `subcategory` 主導で扱い、shape は画面上で表示しない
- `other` も shape を別表示せず、種類選択の続きとして扱う

## 今後の候補

- 将来、帽子や眼鏡など一部種類で同一種類内の型差が必要になった場合にだけ shape を再導入する

## 要再判断

- どの種類で shape を復活させる価値があるかは利用実態を見て再判断する
