# インナー / ルームウェアの責務

## 現状

- `inner` は top-level category として維持する
- UI 表示は `ルームウェア・インナー`
- `underwear` は `inner` 配下ではなく、top-level category `underwear` として分離した

## `inner` に含めるもの

- 見せてもよいインナー
- ルームウェア
- パジャマ

例:

- 見せるキャミ
- ブラトップ
- ルームウェア

## `underwear` へ分離したもの

- ブラ
- ショーツ
- 補正下着
- 見せない肌着

## 判断基準

- 通常のアイテム一覧に出したいか
- コーディネートや着用履歴で、他カテゴリと同じ粒度で扱いたいか

通常一覧に出したいなら `inner`、通常一覧に出したくないなら `underwear` を選ぶ。

## 補足

- 旧来の `inner.underwear` は legacy 分類として読み替え対象になり得る
- 新規登録では `underwear` top-level category を使う
